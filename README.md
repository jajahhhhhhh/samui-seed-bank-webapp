# Samui Seed Bank - Web App

A complete web application clone of samuiseedbank.com - Thailand's premier medical seed bank.

## Project Structure

```
samui-seed-bank-webapp/
├── index.html      # Main HTML structure
├── styles.css      # Complete CSS styling
├── app.js          # JavaScript functionality
README.md       # This file
```

## Features

- Hero Carousel - Auto-playing banner with navigation
- Brand Showcase - Scrollable carousel of seed brands
- Product Grid - 46+ products with category filtering
- Shopping Cart - Add/remove items, saved in localStorage
- Live Search - Product search (Ctrl+K)
- Toast Notifications - Cart feedback
- Responsive Design - Mobile-friendly
- WhatsApp Chat Integration
- Newsletter Signup Form

## Product Categories

- Feminized Seeds (Samui Seed Bank)
- Conscious Genetics
- Perfect Tree Seeds
- Regular Seeds

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript
- CSS Grid & Flexbox layouts
- IntersectionObserver for scroll animations
- LocalStorage for cart persistence

## Running Locally

```bash
git clone https://github.com/jajahhhhhhh/samui-seed-bank-webapp.git
cd samui-seed-bank-webapp
open index.html
```
